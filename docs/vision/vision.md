# Visión del Producto – Emprende UIDE

La visión de Emprende UIDE es crear una plataforma digital (sistema web y aplicación móvil) que funcione como un mercado interno para la comunidad universitaria UIDE, con el fin de que los estudiantes puedan publicar, promocionar y administrar sus emprendimientos de manera sencilla y segura.

El sistema tiene como objetivo fomentar el espíritu emprendedor, facilitar la comunicación entre los estudiantes emprendedores y sus clientes, y centralizar en un solo espacio la oferta de productos y servicios universitarios mediante una plataforma accesible, intuitiva y moderna.
